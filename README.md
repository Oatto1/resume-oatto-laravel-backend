# Oatto Portfolio (Study Case Project)

## Overview

This is a **Full-Stack Study Case Project** built to practice real-world development workflow using Laravel, Filament, and Flutter.

The system allows managing portfolio data from an Admin Panel and delivering it via API to a Flutter mobile application.

---

## Tech Stack

**Backend**

* Laravel
* Filament Admin Panel
* MySQL
* REST API (JSON)
* Spatie Permission (Role System)
* Docker (Laravel Sail)

**Mobile**

* Flutter
* Provider (State Management)
* HTTP API Integration

**Tools**

* Cloudflare Tunnel (HTTPS testing for mobile API)

---

## Features

* CRUD Management
* Portfolio Management (App / Website)
* API Key Protected API
* Role Based Access (Viewer / Super Admin)
* Flutter Mobile App / Laravel API Json

---

## Deployment

* Backend deployed using Railway
* Used for hosting Laravel API and Database in production environment

---

## Purpose

Built as a **Case Study project**:

* Full Stack Development
* API Design
* Mobile + Backend Integration
* Admin CMS Architecture
